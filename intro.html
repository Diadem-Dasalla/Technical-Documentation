Introduction to Cascading Style Sheet (CSS)

What is CSS?

  1. 1. CSS stands for Cascading Style Sheets

  2. 2. CSS describes how HTML elements are to be displayed on screen, paper, or in other media

  3. 3. CSS saves a lot of work. It can control the layout of multiple web pages all at once

  4. 4. External stylesheets are stored in CSS files

Why use CSS?

  1. CSS is used to define styles for your web pages, including the design, layout and variations in display for different devices and screen sizes.

CSS Solved big problems

  1. HTML was NEVER intended to contain tags for formatting a web page!

  2. HTML was created to describe the content of a web page.

CSS Saves a Lot of Work

  1. The style definitions are normally saved in external .css files.

  2. With an external stylesheet file, you can change the look of an entire website by changing just one file!
